## Leetcode: 14. Longest Common Prefix

[https://leetcode.com/problems/longest-common-prefix](https://leetcode.com/problems/longest-common-prefix)

여러 단어들이 배열로 주어질 때, 그 단어들의 공통 접두사를 찾아내는 문제, 예를들어 `["flower","flow","flight"]` 의 공통 접두사는 `"fl"` 이 됨

다양한 풀이방법이 있으나, 위 단어들을 trie 구조에 담은 다음, 찾아내면 됨

```rust
// rust

use std::collections::HashMap;

struct TrieNode {
    end: bool,
    children: HashMap<char, Self>,
}

impl TrieNode {
    fn new() -> Self {
        Self {
            end: false,
            children: HashMap::new(),
        }
    }
}

impl Solution {
    pub fn longest_common_prefix(strs: Vec<String>) -> String {
        let mut trie = TrieNode::new();

        for str in strs.into_iter() {
            let mut n = &mut trie;
            for x in str.chars() {
                n.children.entry(x).or_insert(TrieNode::new());
                n = n.children.get_mut(&x).unwrap();
            }
            n.end = true;
        }

        let mut n = &trie;
        let mut r = "".to_owned();

        while !n.end && n.children.len() == 1 {
            let v = *n.children.keys().next().unwrap();
            r.push(v);
            n = n.children.get(&v).unwrap();
        }

        return r;
    }
}
```

trie 루트부터 시작해서, 자식 노드가 1 개 이면서 (즉 유일하면서), trie 에 포함된 어떤 단어의 끝이 아닐 때까지 탐색하면 됨
