## 테스트

## 🚀 마크다운 렌더링 테스트 페이지

이 페이지는 CSS 스타일링과 렌더링 엔진의 성능을 테스트하기 위해 제작되었습니다.

---

## 1. 텍스트 스타일 (Text Styles)

일반 본문 텍스트입니다. 여기에는 **굵은 글씨(Bold)**, *기울임 꼴(Italic)*, 그리고 **_둘 다 적용된 꼴_**이 포함됩니다.  
또한 ~~취소선(Strikethrough)~~과 `인라인 코드(Inline Code)`가 본문 내에서 어떻게 어우러지는지 확인하세요.

> **참고:** 인용구(Blockquote) 내에서의 텍스트 정렬과 왼쪽 보더(Border) 스타일을 체크하세요.

---

## 2. 목록 스타일 (Lists)

### 순서 없는 목록 (Unordered)
* 1단계 항목
    * 2단계 항목 (들여쓰기 확인)
        * 3단계 항목
* 다시 1단계 항목

### 순서 있는 목록 (Ordered)
1. 첫 번째 과정
2. 두 번째 과정
3. 세 번째 과정

### 체크박스 (Task Lists)
- [x] 완료된 작업 스타일
- [ ] 진행 중인 작업 스타일
- [ ] 대기 중인 작업

---

## 3. 코드 블록 (Code Blocks)

언어별 하이라이팅과 배경색, 폰트(D2Coding, Fira Code 등)를 테스트합니다.

```python
import os

def check_wsl_status():
    """WSL 환경을 확인하는 함수"""
    status = os.uname()
    print(f"시스템 정보: {status}")
    
    for i in range(5):
        print(f"테스트 루프 {i}번 작동 중...")

check_wsl_status()
```
